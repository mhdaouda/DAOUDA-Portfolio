Politique de confidentialité

Cette politique de confidentialité décrit comment les données personnelles sont collectées, utilisées et protégées sur le site www.daoudayinde.com.

Cookies et données de navigation

Ce site utilise Google Analytics pour collecter des données anonymes sur la fréquentation du site. Ces données sont utilisées uniquement à des fins statistiques, afin d’améliorer l’expérience utilisateur.

Un bandeau vous informe de l’usage des cookies à votre première visite. Vous pouvez les refuser ou les accepter librement.

Données personnelles

Aucune donnée personnelle (nom, adresse email, etc.) n’est collectée automatiquement via ce site. Si vous me contactez par email, vos informations ne seront utilisées que pour répondre à votre demande.

Hébergement

Ce site est hébergé via GitHub Pages. Aucune donnée n’est stockée directement sur ce site.
